NOISE & TIMESTEPS

LOSS

MASKED TRAINING

levers_masked_* + mask_cache_dir (config-reader consumed; backends without the lever ignore it)

LAYER FILTER

Target layer preset + explicit filter (config-reader consumed; lora_target_preset family)
